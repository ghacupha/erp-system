# Prepayment Reconciliation Reports

## Unallocated Prepayment Accounts

Use the search reports panel and search for `Unallocated Prepayment Accounts`.

The report lists prepayment accounts where the recognised prepayment amount is greater than the total active amortisation amount by at least `1.00`. It includes the catalogue number, particulars, dealer, debit account, transfer account, currency, original amount, amortised amount, outstanding amount, number of active amortisation entries, and the latest amortisation date.

Use this report when prepayment schedules, ledger balances, or migrated data suggest that a prepayment has not been fully consumed.
