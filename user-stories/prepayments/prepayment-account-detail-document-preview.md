# Prepayment account document review

## Persona

A finance user who is validating a prepayment account before starting or correcting amortisation work.

## Story

1. Open a prepayment-account record from the list or from a report.
2. Review the account details in the grouped detail layout.
3. Select one of the related business document links.
4. Read the scanned PDF in the inline preview on the same page.
5. Compare the source document with the settlement and account values before editing.

## Result

The user can inspect the supporting documents for the prepayment account without opening a separate browser tab or leaving the record view.
