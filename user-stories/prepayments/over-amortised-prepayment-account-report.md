# Over-amortised prepayment accounts

## Persona

A finance reviewer checking whether prepayment amortisation activity has exceeded the original recognised amount.

## Story

1. Open the report search panel.
2. Search for the over-amortised prepayment account report.
3. Open the report page.
4. Review the list of accounts where amortisation has gone past the recognised value.
5. Use the account id or title to investigate the source postings.

## Result

The reviewer can quickly find prepayment accounts that need correction, even when the issue is not tied to a particular reporting period.
