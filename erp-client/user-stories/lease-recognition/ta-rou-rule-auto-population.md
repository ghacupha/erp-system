# TA ROU Recognition Rule Auto-Population User Story

## Persona
Lease accountant configuring ROU recognition rules for a lease portfolio.

## Scenario
The accountant wants the debit and credit accounts to be filled automatically when they
select an IFRS 16 lease contract so that the rule matches the lease template settings.

## Steps
1. Navigate to the TA Recognition ROU Rule create or edit screen.
2. Select a lease contract in the Lease Contract field.
3. Observe that the Debit and Credit account fields update automatically if the
   selected lease contract has a lease template with ROU recognition accounts.

## Expected Outcome
The debit and credit account selections match the lease template's ROU recognition
accounts without additional manual input.
