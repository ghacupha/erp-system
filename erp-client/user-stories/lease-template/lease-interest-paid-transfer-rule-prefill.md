## User story: Prefill interest paid transfer rule accounts from a lease template (client)

**Persona:** Lease accountant configuring transfer rules.

**Goal:** Automatically reuse debit and credit accounts from the selected lease’s template when setting up a rule.

**Flow:**

1. Navigate to the *Interest Paid Transfer Rule* form in the client UI.
2. Choose a lease contract that is associated with a lease template.
3. The debit and credit account controls fill in with the template’s interest paid transfer accounts.
4. If needed, select alternative accounts to override the defaults.
5. Finish any remaining fields and submit the rule.

**Outcome:** Default values from the lease template speed up configuration without preventing manual adjustments.
