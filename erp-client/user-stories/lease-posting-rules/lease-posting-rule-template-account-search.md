# Lease posting rule template account search

## Persona
Lease accounting analyst configuring posting rules for lease events.

## Steps
1. Open **Ledgers → Posting Rules → Lease Posting Rule Configuration**.
2. Select the **Event Type** and **Lease Contract**.
3. In **Posting Rule Templates**, type into the **Debit Account** field to filter the available transaction accounts.
4. Select the debit account and repeat the same for the **Credit Account** field.

## Expected result
- The debit and credit account fields filter results as the analyst types.
- Account categories update automatically based on the selected accounts.
- Both account fields remain on the same row for easy verification.
