# Lease Posting Rule Configuration (Client)

## Persona
Lease accountant configuring contract-level posting accounts.

## Steps
1. Open the Lease Posting Rule Config page from the navigation menu.
2. Choose a lease contract in the form.
3. Verify that debit/credit account suggestions are applied.
4. Review the rule condition for the lease contract ID.
5. Save the configuration.

## Expected Result
The client submits a complete posting-rule payload containing rule metadata, templates, and conditions for the selected lease contract.
